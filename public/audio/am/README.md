# Amharic number call-outs

Place 75 voice clips named `1.mp3` … `75.mp3` here — one per bingo number,
spoken in Amharic (e.g. with the BINGO letter: "ቢ-አስራ አምስት" for B-15).

If a clip is missing, the app plays a short tone instead, so the game still has
audible feedback. Recommended: mono, ~0.6–1.0s, small bitrate.
